# Go API

A simple Go-based RESTful API project.

## Features

- RESTful endpoints
- Modular project structure
- Easy to extend

## Getting Started

### Prerequisites

- [Go](https://golang.org/dl/) 1.18 or higher

### Installation

```sh
git clone https://github.com/yourusername/go-api.git
cd go-api
go mod tidy
```

### Running the API

```sh
go run main.go
```

The API will start on `http://localhost:8080`.
